## 1. 产品概述
Describely AI是一款AI商品描述生成器SaaS工具，旨在帮助商家快速、高效地生成高质量、多语言的商品描述。通过提供便捷的商品信息输入和智能的描述生成功能，解决商家在商品文案创作上的痛点，提升商品吸引力和销售转化率。目标用户是电商卖家、品牌方和营销人员。

## 2. 核心功能

### 2.1 用户角色
| 角色 | 注册方式 | 核心权限 |
|------|---------------------|------------------|
| 普通用户 | 邮箱注册 | 浏览、使用基础功能、商品描述生成（免费3次/月，付费无限次） |

### 2.2 功能模块

1.  **登录/注册系统**:
    -   登录：邮箱 + 6位验证码（输入邮箱 -> 点击"发送验证码" -> 输入验证码登录）。
    -   注册：邮箱 + 验证码 + 设置密码。
    -   页面美观，支持登录/注册切换。
    -   验证码模拟：输入"123456"即可通过（前端模拟）。

2.  **新用户免费次数**:
    -   新注册用户默认拥有3次免费生成机会。
    -   每次生成后，剩余次数减少1。
    -   在页面顶部或工具区域显示剩余次数（例如："剩余生成次数：2/3"）。
    -   免费次数用完后，点击生成按钮弹出"升级提示"弹窗，引导用户付费。

3.  **首页（Tool）**:
    -   **导航栏**: Logo（Describely AI）、Tool、Pricing、My Account。
    -   **右上角**: 语言切换（EN/中文/日本語）、用户头像下拉菜单（个人中心/退出）。
    -   **工具区域（左右分栏）**:
        -   **左侧输入**:
            -   Product Name（商品名）：文本输入框，带placeholder。
            -   Platform（平台选择）：下拉选择框，例如：Amazon, Shopify, Taobao。
            -   Category（类目选择）：下拉选择框，例如：Electronics, Clothing, Home & Kitchen。
            -   Key Features（关键词输入）：文本输入框，支持多关键词，带placeholder。
            -   Tone（语气选择）：下拉选择框，选项：Professional/Casual/Luxury/Persuasive/Minimal。
            -   Output Language（输出语言）：下拉选择框，选项：EN/ZH/JA。
        -   **右侧输出**:
            -   点击"Generate"按钮后，展示3个不同版本的商品描述。
            -   每个版本包含：Title、Description、Bullet Points、SEO Keywords。
            -   每个版本下方有"Copy All"、"Copy Title"、"Copy Description"按钮。
            -   点击Copy按钮后显示"Copied!"反馈。

4.  **使用次数提示**:
    -   在生成按钮上方或旁边显示："Free credits: X/3 remaining"。
    -   次数为0时，Generate按钮不可用，点击弹出升级弹窗。
    -   升级弹窗显示："You've used all free credits! Upgrade to Pro for unlimited access."

5.  **充值/定价页面（Pricing）**:
    -   三个套餐卡片并排展示：
        -   Free：$0/月，3次/月，英文only，基础模板。
        -   Pro：$9.9/月，无限生成，所有语言，SEO优化，5用户。此卡片需突出显示（边框高亮 + "Most Popular"标签）。
        -   Enterprise：$29.9/月，无限生成，所有语言，API接入，不限用户。
    -   点击Pro套餐的"Upgrade Now"按钮弹出支付弹窗。
    -   支付弹窗：显示套餐名和价格、信用卡表单（卡号/有效期/CVC/邮箱）、Pay Now按钮、点击后模拟支付成功。

6.  **个人中心（My Account）**:
    -   顶部显示当前套餐（Free/Pro/Enterprise）。
    -   四个统计卡片：Used This Month、Monthly Limit、Languages Used、Descriptions Saved。
    -   使用历史表格：包含日期、商品名、语言、平台、操作（View按钮查看详情）。
    -   设置区域：
        -   Email Notifications（开关）。
        -   Auto-save History（开关）。
        -   API Key（显示一串密钥，有Copy按钮）。
        -   Dark Mode（开关，控制全局亮色/暗色主题切换）。
        -   Delete Account（红色按钮）。

7.  **多语言切换（i18n）**:
    -   支持English / 中文 / 日本語。
    -   通过右上角下拉菜单进行切换。
    -   所有界面文字（包括提示、按钮、标签、弹窗内容）需随之切换。

8.  **暗色模式**:
    -   个人中心的Dark Mode开关控制全局亮色/暗色主题。
    -   暗色模式配色：背景 #0f172a，卡片 #1e293b，文字 #f1f5f9。
    -   所有页面元素需适配暗色模式。

9.  **使用教程（新手引导）**:
    -   首次登录时，弹出一个简单的3步引导：
        -   Step 1: Enter your product name and keywords。
        -   Step 2: Select platform, tone and language。
        -   Step 3: Click Generate and get 3 optimized versions。
    -   引导弹窗有"Skip"（跳过）和"Next"（下一步）按钮。

10. **其他细节**:
    -   **页面底部Footer**: Copyright 2026 Describely AI，Privacy Policy、Terms、Contact。
    -   **模拟数据生成**: 根据输入的商品名和关键词，前端模拟生成3个不同风格的描述版本（无需真实AI API）。
    -   **交互动画**: 所有交互具备过渡动画，提升用户体验。
    -   **响应式设计**: 适配手机和电脑等不同设备。
    -   **专业蓝色主色调**: 主色调 #2563eb，白色卡片，圆角设计。
    -   **输入框提示**: 所有输入框具备placeholder提示。

## 4. 用户界面设计

### 4.1 设计风格
-   **主色调**: 专业蓝色（`#2563eb`），用于按钮、高亮元素等。
-   **背景色**: 亮色模式下为浅色调，暗色模式下为`#0f172a`。
-   **卡片**: 白色背景，圆角设计；暗色模式下为`#1e293b`。
-   **文字颜色**: 亮色模式下为深色调，暗色模式下为`#f1f5f9`。
-   **字体**: 选用现代、简洁的无衬线字体，保持良好的可读性。
-   **按钮风格**: 圆角、扁平化设计，主按钮使用主色调。
-   **布局风格**: 整体采用卡片式布局，内容区块清晰分隔，导航栏固定在顶部。
-   **图标/表情**: 简洁的线条图标，适当使用emoji增强友好度。

### 4.2 页面设计概览
| 页面名称 | 模块名称 | UI 元素 |
|-----------|-------------|-------------|
| 登录/注册 | 表单区域 | 邮箱输入、验证码输入、密码输入、提交按钮、切换链接、提示文字 |
| 首页 (Tool) | 导航栏 | Logo、菜单项、语言切换、用户头像 |
|           | 输入区 | 商品名、平台、类目、关键词、语气、输出语言选择器，输入框、Generate按钮 |
|           | 输出区 | 3个描述版本卡片，每个含Title、Description、Bullet Points、SEO Keywords，Copy按钮组 |
| Pricing   | 套餐卡片 | Free/Pro/Enterprise卡片，价格、特性列表，Upgrade Now按钮 |
|           | 支付弹窗 | 套餐信息、信用卡输入框、Pay Now按钮 |
| My Account| 顶部信息 | 当前套餐显示 |
|           | 统计卡片 | Used This Month、Monthly Limit、Languages Used、Descriptions Saved |
|           | 历史表格 | 表头、数据行、View按钮 |
|           | 设置区 | 开关组件、API Key显示、Copy按钮、删除账户按钮 |
| 弹窗      | 升级提示 | 标题、提示文字、Upgrade Now按钮 |
|           | 新手引导 | 步骤指示、Next/Skip按钮、内容区 |
| Footer    | 底部链接 | Copyright信息、Privacy Policy、Terms、Contact |

### 4.3 响应式设计
-   采用桌面优先（Desktop-first）的设计策略，再逐步适配移动设备。
-   所有页面元素、布局和字体大小应在不同屏幕尺寸下保持良好显示。
-   针对触摸屏设备进行优化，确保按钮和可点击区域足够大，操作流畅。

### 4.4 暗色模式配色方案
-   **背景**: `#0f172a`
-   **卡片/模块背景**: `#1e293b`
-   **文字颜色**: `#f1f5f9`
-   **主色调元素**: 保持`#2563eb`不变，或进行微调以在暗色背景下保持对比度。
-   **图标**: 适应暗色背景调整颜色。
